# Roadmap

## Near Term

- add operator role model and scoped endpoint access
- expose incident list/detail endpoints
- add richer notification preferences and read/ack workflow
- add retry backoff policies and dead-letter handling

## Mid Term

- OpenTelemetry tracing across API, worker, and realtime services
- dashboards for queue depth, processing latency, and incident MTTR
- idempotency keys for command endpoints
- websocket replay endpoint for late-joining clients

## Long Term

- multi-tenant partitioning
- policy-driven alert correlation
- production deployment manifests for Kubernetes
